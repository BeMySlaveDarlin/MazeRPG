## Requirements
 - NGINX 1.17+
 - PHP 7.3+ with ext:
    - Phalcon 3.4+
    - Swoole 4.3+
    - Memcached 
    - Imagick 
    - DS
 - Composer 1.9+
 - MySQL 5.7+
 - Redis Server 5.0+
 - Memcached Server 1.5+
